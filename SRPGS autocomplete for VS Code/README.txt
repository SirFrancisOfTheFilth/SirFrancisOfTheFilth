Are you a scripter or willing to learn the ancient magic? Do you hate having to type out stuff? Are you perhaps even using Visual Studio Code already or contemplating switching?

Then this guide might be helpful to you, because it shows you how to set up autocomplete for SRPG Studio's script, API and your whole plugin folder.

1.) First off, get VS Code, obviosuly. 
2.) (Optional) Install the prettier extension, which auto formats your code. Not needed but nice to have.
3.) Download all the files from this folder. It should be AutocompleteScript.7z, jsonconfig.json, type-definitions.js and optionally .prettierrc, if you want my prettier config.
4.) Unpack the AutocompleteScript.7z and put that folder into your project folder (The one with the YOURGAMENAME.srpgs file).
5.) Move the rest of the files into your project folder as well.
6.) Start VS Code (if not already done so) and open your project folder with Ctrl + K + O

Congratulations, you now have autocomplete for the entire SRPG Studio script, API and everything in your plugin folder!
